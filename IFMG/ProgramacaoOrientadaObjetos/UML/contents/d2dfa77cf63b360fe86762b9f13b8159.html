Description

First fill the circle buffer with as many characters as are in the to string. If we reach an early end, bail. We will loop, possibly for all of the remaining characters. Compare the circle buffer with the to string. If we exit the loop with b intact, then victory is ours. Get the next character. If there isn't one, then defeat is ours. Shove the character in the circle buffer and advance the circle offset. The offset is mod n.
